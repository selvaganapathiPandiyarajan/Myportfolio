Clone Red Bus

Project Descripition

Login Page: I developed a Login Page for the RedBus clone using Angular and Node.js, allowing users to log in with either their email or phone number. The system securely validates the credentials via API requests to the backend, ensuring a seamless and flexible login experience for users.

Sign-Up page: I Developed a Sign-In Page using Angular with input fields for first name, email, phone number, gender, and GST number, including form validation. The data is securely submitted and stored in PostgreSQL through Node.js, ensuring proper validation and storage of user details.

Home Page: I Created a Home Page for the RedBus clone with details about available bus services. Users can select their travel details, including the "From" and "To" destinations, and choose their travel dates using an intuitive date picker for easy trip planning.

Bus Details Page: I developed a Bus Details Page for the RedBus clone where users can view comprehensive bus information, including features such as seater, sleeper options, free Wi-Fi, toilet availability, cost, boarding points, ratings, and bus photos. The page includes filter functionality, allowing users to narrow their search based on preferred bus features.

Ticket Booking Section: I developed a Ticket Booking Section for the RedBus clone, where users can select their seat number, boarding stop, and destination stop. After selecting these details, users can review their booking information on a confirmation page to ensure all chosen details are correct before proceeding with the booking.

Ticket Booking Section: I created a Payment Section for the RedBus clone using Razorpay, allowing users to securely process payments for their bookings. The integration supports multiple payment methods, including credit/debit cards, UPI, and net banking, providing a seamless and secure transaction experience.
