RBWO Tool

Project Descripition

Holiday upload Section I created a Calendar Upload Page in Angular that allows users to upload Excel files, storing event data in local storage and organizing it by year for easy management.

User Manual Section I developed a User Manual Page that provides instructions on how to use the app, including templates for project management and holiday requests, offering users clear guidance and resources for effective application utilization.

Project activity upload Section I created a Project Upload Page that allows users to upload Excel files containing project descriptions, start dates, and end dates, enabling easy import and management of project data within the application.

Project Output Generate Section I developed a Project Output Page that generates project reports based on acceptance criteria, including a specified day range for Quality Assurance (QA) and buffer periods, allowing users to visualize project timelines and track progress effectively.

Project Output download Section I implemented an Output Download Section that allows users to download an Excel report calculating effort on a month-wise basis, including Full-Time Equivalent (FTE) and time metrics, along with visual graphs for Project Time Estimate (PTE), Project Performance Metrics (PPM), milestones, and deliverables.
