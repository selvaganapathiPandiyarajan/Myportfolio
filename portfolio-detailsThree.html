TicketMangament System for Support Team

Project Descripition

Login Page: I developed a Login Page for the RedBus clone using Angular and Node.js, allowing users to log in with either their email or phone number. The system securely validates the credentials via API requests to the backend, ensuring a seamless and flexible login experience for users.

Sign-Up page: I Developed a Sign-In Page using Angular with input fields for first name, email, phone number, gender, and GST number, including form validation. The data is securely submitted and stored in PostgreSQL through Node.js, ensuring proper validation and storage of user details.

Forgot Password: I developed a Forgot Password Page using Angular and integrated EmailJS to send an OTP to the user's email for secure password recovery. After entering the registered email, the user receives a one-time password (OTP) via email, which they can use to verify their identity and reset their password securely.

Customer Dashboard: I developed a Customer Dashboard using Angular and Node.js, featuring a ticket creation system, profile page, password reset functionality, and an edit profile section. The dashboard includes advanced features like filtering and pagination to manage and display tickets efficiently, ensuring a smooth user experience for handling support queries and personal information.

Manager Dashboard:

I developed a Manager Dashboard using Angular and Node.js, featuring team management capabilities, where all tickets from customers can be assigned to team members. The dashboard includes functionalities for editing profiles, managing profile pages, resetting passwords, and tracking ticket statuses with sections for completed and in-progress tasks, enhanced by progress bars. Additionally, the manager can delete team members as needed, ensuring efficient team and ticket management.

Support Team Dashboard: I developed a Support Team Dashboard using Angular and Node.js, where team members can view tickets assigned by the manager, review their details, and close tickets once resolved. The dashboard features an edit profile section and a password reset page, along with filtering and pagination options for efficient ticket management. It includes distinct sections for tracking progress and completed tickets, providing a clear overview of ongoing support activities.
