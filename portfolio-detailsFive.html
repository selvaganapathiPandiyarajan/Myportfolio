S'Traval

Project Descripition

Login Page: I developed a Login Page for the S'Traval using Angular and Node.js, allowing users to log in with their email. The system securely validates the credentials via API requests to the backend, ensuring a seamless and flexible login experience for users.

Flight Dashboard I developed a Flight Booking System using Angular and Node.js that allows users to search and book flights with real-time pricing, country and city-based filtering, and a seamless user workflow. The module features a fully interactive calendar that displays daily prices, enabling users to easily compare fares across different travel dates and choose the best option. Users can refine results using filters such as travel origin, destination, country selection from the navigation bar, and travel date ranges, providing a more personalized and accurate search experience. The system displays detailed flight information including airline, timing, route, and fare, and supports smooth navigation from search results to booking confirmation. This feature-rich flight module delivers a clean, modern UI and replicates the experience of professional airline ticket platforms.

Hotel Dashboard: I developed a Customer Dashboard using Angular and Node.js, featuring a ticket creation system, profile page, password reset functionality, and an edit profile section. The dashboard includes advanced features like filtering and pagination to manage and display tickets efficiently, ensuring a smooth user experience for handling support queries and personal information.
