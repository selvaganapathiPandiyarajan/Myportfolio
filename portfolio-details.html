TimeSheet Management

Project Descripition

Login Page:I developed a secure login page using Angular,Node js,Pgsl with a "Remember Me" feature that stores user sessions in local storage for persistent login.

TimeSheet:I developed a TimeSheet Management system using Angular,Node js,Pgsl integrated with FullCalendar for seamless scheduling, allowing users to log hours, track tasks, visualize time entries in a calendar view and have upcoming events.

Employee referral:I developed an Employee Referral Management system using Angular,Node js,Pgsl enabling employees to refer candidates for open positions, track the status of referrals, and manage rewards for successful hires. The system streamlines the referral process with an intuitive interface, automated notifications, and real-time updates on the progress of referred candidates.

Hours History:I Implemented an Hours History feature in the TimeSheet Management system using Angular,Node js,Pgsl enabling users to view and track their logged hours over time, with detailed entries displayed in a table or calendar format for easy reference and review.

Leave Management:I developed a Leave Management module in the TimeSheet Management system using Angular,Node js,Pgsl allowing users to apply for leave, track leave balance, and view leave history. Integrated with FullCalendar for an intuitive display of leave requests and approvals, ensuring smooth coordination between employees and managers.

Holiday Management:I created a Holiday Management system using Angular, allowing users to view and manage holidays within the organization. The system includes features for tracking public holidays providing an intuitive interface for users to check their holiday balances and submit requests, all while ensuring seamless integration with existing leave management processes.
